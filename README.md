# 机房助手 - ITClassHelper

# 简介
这是一款帮助你对抗老师的"电子教室"而开发的软件，拥有关闭、挂起、恢复、读写密码、控制广播等功能。\
同时它还可以向机房里的任何设备远程发送命令，让你成为机房里最靓的仔！
本软件面向"极域电子教室"开发，并从 3.2.0 版本后开始逐渐支持"红蜘蛛电子教室"，以后将会支持更多电子教室。

# 功能
* 实时显示教室状态
* 关闭、挂起、恢复教室（仅极域）
* 读取/设置教室密码（仅极域）
* 一键补全并更新组件
* 自动读取教室路径（仅极域）
* 破解教室各种挂钩（仅极域）
* 设备管理器：
	* 扫描在线设备
	* 可发送命令、消息、脚本
	* 远程关机、重启
	* 内置脚本处理器：将BAT/VBS转换为可攻击脚本

# 安装

直接下载仓库中 Release 文件夹中的 ITCHLauncher.exe 文件，打开后将自动下载必要组件并打开主程序。\
