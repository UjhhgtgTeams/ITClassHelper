# 机房助手 - ITClassHelper

# 介绍
这是一款帮助你对抗老师的"极域电子教室"而开发的软件，拥有关闭、挂起、恢复、读写密码、控制广播等功能。\
同时它还可以向机房里的任何设备远程发送命令，让你成为机房里最靓的仔！

# 功能
* 实时显示极域状态
* 关闭、挂起、恢复极域
* 读取/设置极域密码
* 一键补全并更新组件
* 自动读取极域路径
* 机房攻击：
	* 可发送命令、消息、脚本
	* 内置脚本处理器：将BAT/VBS转换为可攻击脚本

# 安装
## 初次安装
直接下载仓库中 Release 文件夹中的 ITClassHelper.exe 文件，打开后再点击[更新软件]即可。\
[点我立即下载](https://gitee.com/ujhhgtg/ITClassHelper/raw/master/bin/Release/ITClassHelper.exe)
